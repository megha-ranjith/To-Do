## Installation

1. Clone repository
```sh
git clone https://github.com/megha-ranjith/flask-todo-app.git
cd flask-todo-app
```

2. Create virtual environment
```sh
python -m venv venv
venv\Scripts\activate # Windows
```
```sh
source venv/bin/activate # macOS/Linux
```

3. Install dependencies
```sh
pip install -r requirements.txt
```

Run app
```sh
python app.py
```
